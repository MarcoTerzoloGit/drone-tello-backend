# DJI TELLO DRONE SERVICES

Node js application to handle drone commands, get flight data get and video stream.

Exposes websocket communication to be interconnected to the DRONE-TELLO-FRONTEND project.

This project, inspired by other projects already started in this direction aims to do some basic personal research on drone data usage and management, computer vision, 3d visualization in javascript and autonomous drone flying using on board internet connection.

It's open source so any part of it can be used as your own advantage. Feel free to fork it and share your results
